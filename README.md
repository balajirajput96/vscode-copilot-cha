## 🎯 HashiCorp Boundary Automation Suite

A complete automation suite for HashiCorp Boundary with HCP integration is available in the [`boundary-automation/`](boundary-automation/) directory.

**Quick Start:**
```bash
cd boundary-automation
./auto-connect.sh
```

**Features:**
- 🚀 Automated Boundary CLI installation
- ⚙️ HCP configuration wizard
- 🎯 Target management (SSH, HTTP, Database)
- 🐳 Docker test environment with 6 services
- 📚 Comprehensive documentation

See [`boundary-automation/README.md`](boundary-automation/README.md) for full documentation.
